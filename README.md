# Auction_House

# USER MANUAL:

To see the funtionlaity of the django based web application :

clone this repo using, git clone 'https://github.com/AyushBasera/Auction_House'

then navigate to the project directory by using, 'cd Auction_House'

then execute the command in the same directory to runserver, python manage.py runserver

open browser and visit this link, http://127.0.0.1:8000/

Now we can explore the web app by registering on it.

press ctrl + c to stop the server.

# ABOUT THE PROJECT:

A full-stack online bidding platform using Django for backend logic, SQLite for databases, and HTML/CSS with Bootstrap for a responsive frontend interface.

Enabled hosts to create auction listings and users to place bids on listed items.

Implemented auction closure logic to sell items to the highest bidder when the host ends the auction.
